# Foundational Cloud Infrastructure via Terraform

This repository orchestrates a production-ready, clean cloud architecture stack on AWS utilizing HashiCorp Terraform.

## Architectural Layout Features
* Complete VPC Segregation Isolation
* Dual Layout Multi-AZ Subnet Splitting
* Stateful Network Security Group Port Filtering
* Automated Dynamic Machine Image Searching Matching

## Execution Run Sequences
```bash
terraform init
terraform plan
terraform apply --auto-approve
\`\`\`
